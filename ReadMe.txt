If the system failed to installed, there might be problem
1. missing vcredist2005, in that case install both packages in the vcredist
2. directx missing, in that case comment the direct line in the plugins.cfg file

Required Libraries:
1. boost 1.3.5 library
2. ogre 1.6.2

Important !
The updated vcredist_x86.exe is found here:  C:\Program Files\Microsoft Visual Studio 8\SDK\v2.0\BootStrapper\Packages\vcredist_x86
